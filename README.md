# Process-Automation-Dashboard

Automates **repetitive reporting workflows** using a mock operations dataset (10,000 rows of orders, delays, costs).  
One-click ingestion → cleaning → KPI calc → PDF/Excel export.  
Deploy in **60 seconds** locally or on **Streamlit Cloud** (free).

---


## What It Automates (10+ hrs saved)
| Manual Task | Old Time | New Time | Automation |
|-----------|---------|----------|------------|
| Excel data dump → clean | 4 hrs | 8 sec | Pandas one-liner |
| Pivot tables (region, delay) | 3 hrs | 3 sec | Auto-groupby |
| Charts + annotations | 2 hrs | 2 sec | Plotly + Streamlit |
| PDF report email | 1.5 hrs | 0 sec | Auto-download |
| **Total** | **10.5 hrs** | **< 15 sec** | **42x faster**
